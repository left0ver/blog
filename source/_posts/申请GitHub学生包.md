# 我遇到的一些坑

1. 学生证的首页信息上没有写学校名称，盖了一个学校的章，所以建议大家使用学信网的在线验证报告申请，并且是英文的

2. 申请的时候不能使用VPN

3. 上传信息的时候，使用手机拍照上传，不要选择文件上传（成功率低）

4. GitHub的profile中的姓名改为自己的中文名字的拼音

5. GitHub的[billing](https://github.com/settings/billing/payment_information) 中的信息需要改为自己中文名字的拼音

# 使用学信网申请

1. 打开学信网，申请一份在线验证报告，下载下来，然后使用[deepl](https://www.deepl.com/zh/translator/files) 将其翻译成英文
2. GitHub的profile中的姓名改为自己的中文名字的拼音
3. GitHub的[billing](https://github.com/settings/billing/payment_information) 中的信息需要改为自己中文名字的拼音
4. 关掉VPN
5. 在手机上打开[GitHub Education benefits](https://education.github.com/discount_requests/application) 填写信息进行申请(最好使用教育邮箱，不过在网上看到有人没用教育邮箱也申请成功了)，拍照上传英文的在线验证报告。
